The Woo better load spinner will allow you to modify the loading screen or transcional spinner of Woo, for a better user experience with your customers.
== Description ==

= Summary =

The plugin is activated on your site automatically after being installed, automatically replacing the default loading spinner of Woo, by one more showy and especially sticky, much easier to see than the base. Thus allowing users less used to online stores not to get stuck while performing a search in the product database or similar. 


= Features =

* Mejora la experiencia de usuario de los visitantes de tu web, gracias a este ligero plugin, que transforma la transición predeterminada de woocomerce en una mucho más visible y animada. 

== Installation ==

1. Install Woo better load spinner from the WordPress Plugin Directory. Or, if that doesn't work:
2. Upload the "Woo better load spinner" directory to your "wp-content/plugins" directory.
3. In your WordPress admin, go to PLUGINS and activate "Woo better load spinner"
4. Everything is done!
